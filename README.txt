# Portfolio

### Description
Ce dépôt contient mon portfolio personnel, un projet évolutif qui recense les exercices de mise en pratique réalisés au fil de mes cours en développement web.  
Je commence avec du **HTML**, **CSS** et **JavaScript**, mais je compte intégrer d'autres technologies à l'avenir !

### Installation
Pour cloner et lancer ce projet en local :  

git clone https://github.com/XavierSARAZIN/Portfolio/
cd portfolio

Ou directement via ce lien ! => https://xaviersarazin.github.io/Portfolio/
